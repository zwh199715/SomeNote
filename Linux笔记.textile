Linux学习：

常用快捷键：
  切换历史命令： ctrl+p  ctrl+n
  光标移动： CTRL+F Ctrl+E Ctrl+A  光标向后，光标移动到末尾，光标移动到头部
  删除：光标盖住的是光标后边的字符，光标前面的是当前覆盖的字符前面的字符  Ctrl+u删除光标前面的所有字符，CTRL+d删除光标后面的一个字符
  提示键： tab  两次 tab列举所有类似命令
  A
目录结构：
	/bin目录存了很多常用命令
	/boot目录存了很多开机启动项
	/dev设备文件 一切皆文件，将硬件设备抽象成文件保存在dev中
	/etc保存配置文件
	/home 存放linux下的所有用户的目录
	/lib 存放动态链接库 dll
	/lost+found 文件碎片
	/media  自动识别后挂载生成
	/mnt 两个目录都是用来挂载外部设备的 如 U盘  mnt用户自己挂载 如光驱等
	/opt 第三方软件
	/proc
	/root 系统管理员，超级权限用户的用户主目录
	/sbin 管理员使用的命令
	/usr目录 用户软件资源目录 当前用户装的一些软件

目录的路径：
	相对路径：相对于当前目录

	绝对路径：从根目录开始
	.表示当前目录
	..表示当前目录的上一级目录
	-上一个访问的目录 两个邻近的两个目录切换
	~当前用户的家目录  cd ~就是当前用户的家目录
	$表示普通用户
	#表示root用户
	sudo su 输入当前用户密码 切换到root用户
	exit推出root用户 返回到当前用户

查看文件目录：
	1.用tree软件查看
	2.ls -l  列出本目录下的所有文件和详细信息 详细信息的前11个字符的含义，第一个字符表示文件的类型包括：d目录，-普通文件，l链接符合，b块设备，c字符设备，s socket文件，管道p 其余的分三组  从左到右依次是
	所有者，同组用户，其他用户
	权限包括：rwx 读 写 执行

	权限后面的第一个数字表示 硬连接数，后面就是文件所有者 后面是一个数据  如果文件类型为普通文件 这个数字就是这个文件的大小，如果是目录就是固定的4096不管里面是否有文件 后面就是创建时间 最后是文件名
	drwx-rwx-rwx-2-user-4096-date-filename


	3. ls + 路径 列出该路径下面的文件
	4. ls -la 列出目录下的隐藏文件

文件和目录的创建

 	1.mkdir dirname 创建目录
	2. mkdir -p dirname1/dir2/dir3创建多级目录
	3.touch + 文件名.类型 创建文件 如果不存在创建文件，如果文件存在修改文件创建时间

 文件目录的删除
 	1.rmdir 删除空目录 是彻底删除
 	2.rm -r dirname 递归的删除这个目录和所有下面的文件
 	3.rm -ri 删除时询问

 文件和目录的复制
 	1.cp 要拷贝的文件+新的文件名：如果文件名字已经存在会覆盖原来的内容，如果不存在就创建文件
 	2.cp -r 要拷贝的目录 新的目录名 如果没有则创建 如果存在 就会把拷贝的目录放进存在的目录 名字就是拷贝的目录名

 查看文件的内容
 	1.cat 文件名  将文件内容输出到屏幕上
 	2.more 文件名  按回车或者空格翻页，但是只能往后翻不能往回看 想退出时按q键
 	3.less 文件名 CTRL+F向后翻页 CTRL+B向前翻页
 	4.head 文件命 默认显示文件的前十行 可以跟 head -5 查看前5行 tail 查看后十行

 修改文件的名字
 	1. mv 文件名 文件名1 改名
 	2. mv 文件命 目录名  将文件移动到这个目录下面
 硬链接的创建
 	软链接就像快捷方式
 	1. ln + 文件名 --文件名1  给文件加硬链接 Linux中的文件存在磁盘上是通过inode结点查找的，硬链接就是不同的名字指向同一个文件位置inode节点，那么不管多少个实际的内容都只有一个 如果修改其中一个的内容 那么所有的都会改变
 	2.软链接可以指向硬链接，有的硬链接需要快捷方式有的不需要 目录不能创建硬链接
 	3. ln -s 文件命 软链接名
 文件和目录的属性
 	1.wc 文件名 查看文件的相关信息 字节数 
 	2.od -t 文件名 查看二进制文件的信息  t后面可以有 c ASII字符，d有符号十进制数，f浮点数，o八进制，u无符号十进制，x十六进制
 	3.du 查看某个目录大小 加参数-h 人能看懂的信息human
 	4.df 看磁盘使用情况
 which命令
 	which+命令  显示当前命令所在的目录 查不到的说明是内置命令

查看和修改文件的权限
 	1.文字设定法
 		chmod [who] [+] [-] mode
 		who:
 			u 文件拥有者
 			g 同组用户
 			o 其他人
 			a 所有
 		+增加权限
 		-减少权限
 		=覆盖原来权限
 		mode:
 			r w x
 	2.数字设定法
 		chmod 777 文件名 r:4 w:2 x:1

修改文件所属的组和所有者
 	1.文件可以属于某个用户但是不一定属于这个用户的组！！！
 	2.修改文件的所有者 使用chown命令
 	3.chown +新的所有者：新的所属的组 +文件名 需要使用管理员权限 在名字前面使用sudo
 	4.chgrp 新的所属组+文件名
 	5.目录必须有执行权限 才能进入这个目录

软件的安装和卸载
	在线安装
		sudo apt-get install tree 在线安装
		sudo apt-get remove tree 卸载移除
		sudo apt-get update 更新软件列表（软件的名字和下载地址）
		sudo apt-get update clean 清理所有的安装包---/var/cache/apt/archives
	deb包安装
		sudo dpkg -i xxx.deb
		sudo dpkg -r xxx
	源码安装
		1.解压缩源代码包
		2.进入安装目录
		3.检查文件是否缺失，创建makefile，检测编译环境：。/configure
		4.编译源码，生成库和可执行程序：make
		5.把库和可执行程序，安装到系统目录下：sudo make install
		6.删除和卸载软件：sudo make distclean
		7.上述安装步骤并不是绝对的应该查看附带的README文件
